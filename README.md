# Law_Retriever
A law regularizations provider with ability of legal data and corresponding cases retrieval.

### Codes available below:
* /src-main system codes 
* start.py-starter of law regularizations retriever 
* retriever.py-batches of law cases processing entrance 
* retriever_judge.py-single law case retrieval file 
* web-front end interfaces configuration files 
* train-lora-based refinement files for GLM model 
* retriver-detailed file of retrieval function 
* text2vec-base-chinese-chinese texts conversion to vectors 
* peft_r_model-pickle modules of retrieval 
* peft_model-pickle model files 
* peft-model setup 
* model-chatglm pytorch model and files 
* /data-dataset of law regularizations

### Commanders available:
* python ./start.py
#### Single mode commander:
* python ./retriever.py
* or
* python ./retriever_judge.py
